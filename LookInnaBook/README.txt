To run the code:

1. Make a database on pgadmin4, name it whatever you want
2. Run the sql file COMP3005PROJECT.sql
3. Then run dml.sql to populate the tables with some data

4. download the jdbc driver from https://jdbc.postgresql.org/download/ (download version 42.5.1)
5. Use intelliJ to open the java project, go to file->project structure and go to Project Setting->Libraries, then
	click add and locate the postgresql-42.5.1 jar file that you downloaded.
6. Rename the first 3 variables in main to your liking: database, user and password to match your postgres database.
7. You can now run the code in IntelliJ!
